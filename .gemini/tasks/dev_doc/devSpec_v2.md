# 數位賀卡 V2 現代化 RPG 改版需求書 (Wedding Quest V2)

## 1. 專案背景與核心理念
本專案為送給學妹的專屬結婚數位賀卡。將大學時期「志工服務性社團」的共同回憶（四書五經、南寮/永安淨灘、跨校寶可夢迎新、開伙煮飯）與 J-RPG 視覺小說風格結合。
設計哲學需秉持「可愛就是正義」的日系精緻感，並套用現代網頁前端技術，提升手機端的沉浸式互動體驗。

## 2. 視覺風格與 UI/UX 升級 (Modernization)
* **玻璃擬物化對話框 (Glassmorphism)**
  * 背景使用深色半透明 (`rgba(15, 23, 42, 0.7)`) 搭配高斯模糊 (`backdrop-filter: blur(8px)`，需加 `-webkit-` 前綴)。
  * 邊框加上極細半透明白線與柔和陰影，取代傳統死板色塊。
* **字體混搭美學 (Hybrid Typography)**
  * **對話內文**：使用現代無襯線體（Noto Sans TC / 微軟正黑體），字體顏色 `#F8FAFC`，行距 `1.6`。
  * **角色名稱 / 系統提示**：保留 16-bit 像素字體 (Pixel Font)，並加上文字陰影 (`text-shadow`)。
* **全螢幕觸控熱區與流暢度**
  * 推進對話的點擊事件 (`onClick`) 需綁定於全螢幕遮罩層，方便單手操作。
  * 對話框右下角需有平滑上下浮動的「點擊繼續」Icon（如小箭頭或像素愛心）。

## 3. 角色設定與美術轉場 (Characters & Transitions)
* **角色職業設定 (3頭身日系像素立繪)**
  * **社長 (玩家視角)**：拿著三角板或法杖的「數學魔法師」。
  * **學姐**：拿著書本或教鞭的「教育系補師」。
  * **學妹 (主角)**：初始裝備為「社團 T 恤」的音樂/教育雙修「吟遊詩人」；最終句對話時，觸發轉職特效換上「婚紗裝備」。
* **場景與特效切換**
  * **前半段場景**：南寮/永安漁港的黃昏海岸，背景可點綴腳踏車與史萊姆（垃圾），或是社團開伙的大平底鍋。
  * **轉場特效**：對話進行到「登入婚姻伺服器」時，觸發全螢幕白光閃爍（致敬寶可夢進化/精靈球特效）。
  * **後半段場景**：閃白結束後，背景無縫切換為「華麗的像素風婚禮教堂」。
* **圖片渲染確保**
  * 所有 `.sprite` 或 `.pixel-bg` 必須加上 `image-rendering: pixelated;`。
